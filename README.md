# IEEE 802.11 Security Analyzer (ARM64/x86_64)

A tool for analyzing wireless network security using C++ and system programming.

## Features
- **Low-level:** Privilege checking is written in **ARM64 Assembler**.
- **Performance:** Kernel written in **C++20**.
- **Portable:** Automatic dependency installation.
- **Modern UI:** User-friendly "Polyanka" interface in the terminal.
